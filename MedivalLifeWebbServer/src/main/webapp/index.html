A Medieval Random Heritage generator

A training project

This is a app for generating a dynasty tree (for use as a background of an RPG character, or just to populate a small medieval hamlet) with code for the common heritage software appended at the end. It generates a tex, first describing the living and dead members of the family and then (hidden for use with javascript later) the same info (more or less) in Gedcome code for use in heritage software.
