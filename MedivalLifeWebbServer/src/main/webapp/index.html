A portfoilo of applications

This is the home for my portfoilo of programs I create to learn and amuse myself. Usually they have some application to Roleplaying games (table top) or maybe for authors. The links above goes to their respective wikipedia and Github repositories, where instructions for their use and uptodate status can be found.

StarViewer

An app to generate (fictional) star systems,when finished it will be delivering a cluster of stars and planet systems with complete physical characteristics and also populated with biology (where applicable) and civilisations. Currently only avaiable as runnable jar file (see wiki for details).

Medieval Life Heritage Generator

This is an app for generating a dynasty tree (for use as a background of an RPG character, or just to populate a small medieval hamlet) with code for the common heritage software appended at the end.

Some project in the pipeline

Goblins First Dungeon

A demo made in Unity3D for a game inspired by classical games as "Dungeon Keeper" but with a slant more towards survival/crafting than humorous wargaming. Development is currently on hiatus due to Java intense Java learning.

A "How to Host A Dungeon" implementation

A test of implement the excellent "How to host a dungeon" in a ASCII environment. On the wait list due to decision paralysis: Keep the vertical aspect or redesign it into a fully featured multilevel dungeon.
